1. Create a git repo and clone it in "local"
2. Generate sample data
3. Push new changes to git and initialize dvc.
4. do dvc init (creates .dvcignore, .dvc)
5. Add the data storage for dvc (could be S3, etc), but temporarily, I will add a new directory called "S3", mimicing AWS S3
6. Link dvc to the storage defined in step 5 above. Do "dvc remote add -d <assign a name to the connection> <path to the storage>
    Ex. dvc remote add -d myRemote S3
7. Lets add some data to dvc:
    dvc add <data folder>
    Ex. dvc add data/
    (When you do it the first time when data/ is being tracked by git, it will throw an error. To make it work, remove data/ from getting tracked by git
    git rm -r --cached 'data'
    git commit -m "stop tracking data"
    )
8. dvc add data/ will create data.dvc and populate .dvc
